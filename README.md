这个工程项目管理 Notion 模板包含以下专业功能：
📋 核心字段结构
表格
字段	说明
序号	自动编号，支持排序
日期	项目登记/开工日期
项目名称	工程全称
工程款	合同金额/总收入（绿色显示）
支出	已投入成本（红色显示）
结余	工程款-支出（实时计算）
进展	备案 → 验槽 → 验收（流程化）
备注	详细说明、关键节点
🎯 关键功能
流程化进展管理
三阶段状态：备案（ amber 橙黄）→ 验槽（ blue 蓝）→ 验收（ emerald 绿）
下拉菜单快速切换状态
完工标记（紫色标签），自动置灰并划线
财务看板
项目总数、工程总款、总支出、项目结余四大指标
实时计算单笔项目结余（正数蓝色/负数红色警示）
智能筛选
按进展阶段筛选（备案/验槽/验收）
完工项目单独筛选
项目名称关键词搜索
数据操作
新增/编辑项目（模态框表单）
快捷编辑进展状态
删除确认保护
导出 CSV（兼容 Excel）
交互体验
行悬停高亮+位移效果
完工项目自动透明度降低
键盘快捷键：ESC 关闭弹窗、Ctrl+N 新建
响应式布局，支持移动端查看
